# Prompt Engineering Example

## Weak Prompt
Explain diabetes.

## Model Response (Weak Prompt)
Diabetes is a disease related to blood sugar.

---

## Improved Prompt
Explain diabetes in a clear, structured way for a patient. Include:
- Definition
- Causes
- Symptoms
- Treatment

## Model Response (Improved Prompt)
Diabetes is a chronic condition that affects how your body processes blood sugar (glucose).

Causes:
- Type 1: autoimmune destruction of insulin-producing cells
- Type 2: insulin resistance

Symptoms:
- Increased thirst
- Frequent urination
- Fatigue

Treatment:
- Lifestyle changes
- Medication or insulin therapy

---

## Evaluation

### Clarity
- Weak prompt: vague output
- Improved prompt: structured and clear

### Completeness
- Weak prompt: incomplete
- Improved prompt: comprehensive

### Usefulness
- Weak prompt: low value
- Improved prompt: highly useful for patients

### Final Insight
Better prompt design significantly improves response quality, structure, and usefulness.
